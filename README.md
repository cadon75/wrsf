# WRSF

主にMMLからなどから作製されたソフトウェア音源（FM音源等）サウンドのwavファイルをターゲットとした  
懐かしい感じの音に変換する変換型簡易フィルタです。

## 動作環境 ( 動作確認済環境 )
*   **OS**: Windows 11 (X86_64) / Linux (Ubuntu系)
*   **依存関係**: Linuxでは使用する環境によっては libgtk-3-dev が必要になる事があります。  
ターミナルで操作しますが、GUI(ExplorerやLinuxでは各種ファイラーなど)の機能も使用します。

## 主な使い方 / 機能
*   主に  
「昔の機器からライン入力されたラジカセで録音されたカセットテープから再生してるっぽい音」  
にするのがメインです。(ぽいが重要)
*   変換できるファイルは  
**wavファイル (22.5/44.1/48/96KHz ・ int 16/24/32bit / float 32bit ・ 2ch-Stereo) のみ** です。
*   GUI上のターミナルからバイナリを起動すれば、対話方式で処理方法を選択できます。
*   バイナリにファイル名を付けて起動すれば基本フィルタ処理をすぐに一気に行います。
*   書き出すwavファイルは、上書き確認ありの追記自動命名です。都度ユーザーが対応して下さい。

*   **フィルタ**  
    1. コンシューマーフィルタ  
    レトロな機器の音の特性をwavファイルに簡易的に付与します。  
    2. カセットテープフィルタ  
    カセットテープの音の特性をwavファイルに簡易的に付与します。  
    3. スピーカーフィルタ  
    ラジカセのスピーカーの様な音の特性をwavファイルに2次フィルタで付与します。  

*   フィルタは 44.1KHz - 16bit int - Stereo をメインターゲットにしています。他の周波数対応はオマケです。

*   簡易フィルタですが、中身は結構重い処理もしています。  
現代的な環境なら数秒ですが、読み込むwavファイルや環境によっては処理に時間がかかります。  

* オススメの使い方は、基本的に基本フィルタで大丈夫ですが、  
音がこもり過ぎと思う場合は、コンシューマーフィルタのみ2番に変えてみて下さい。  
お気に入りのレトロなスピーカーなど、特定の視聴機器の特性を活かしたい場合は  
コンシューマー＋カセットテープフィルタのみなどをお試し下さい。

## ビルド
*   リポジトリをクローンしてから、Windows / Linux 上 の RUST (作製時は RUST 1.95.0) で  
それぞれ使用環境に合わせてビルドするだけです。   

```
    Windows : cargo build --release --target=x86_64-pc-windows-msvc
    Linux   : cargo build --release --target=x86_64-unknown-linux-gnu
```    
## 注意事項
* あくまで作者の個人的補助用に作製したソフトです。  
精度や忠実なリアル感は求めないで下さい。  

* 一応、使うフィルタの設定などを出来るように設計していますが  
出来るようにしてあるだけのオマケで検証等はしていません。  

* 動作中は結果を書き出すまでオンメモリ動作です。  
あまりにも巨大サイズwavファイルやメモリが極端に少ない環境ではご注意下さい。

* 個人ソフトなのでSmartScreenなどが反応する可能性が高いです。  
免責の通りなので信用出来ない場合はご利用を控えるか、各環境でビルドしてください。
